# MTL Classifier Optimization (2026-03-02)

This experiment dossier is dedicated to optimizing the AtlasMTL **MTL
classifier** itself, following the conclusions in:

- `documents/experiments/2026-03-01_real_mapping_benchmark/results_summary/experiment_report_2026-03-02.md`

Positioning constraints:

- AtlasMTL is treated as a reliable `sc -> sc reference mapping` and
  multi-level label transfer method.
- Primary endpoints focus on label accuracy, abstention quality, hierarchy
  consistency, robustness, and traceability (not embedding integration
  metrics).

Directory layout:

- `plan/`: execution plan and experiment matrix
- `notes/`: ad-hoc discussions / design notes (keep short and dated)
- `scripts/`: run helpers for this optimization cycle (if needed)
- `results_summary/`: compact summaries pointing to large artifacts under
  `~/tmp/`
